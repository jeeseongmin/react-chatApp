## SignUp & SignIn & SignOut

    1. SignUp (회원가입)
    - SignUp은 단순하게 [Home] - SignUp button 클릭 - SignUp page 로드
    - SignUp page의 view 구조는 Email input, Password input, submit button
    - Firebase에서 createUserWithEmailAndPassword 함수를 사용한다.
